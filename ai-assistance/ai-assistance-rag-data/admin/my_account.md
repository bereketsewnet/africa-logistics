# My Account

## What This Page Is

The My Profile section is the admin's personal account page.

It is used for profile identity, security, and personal account preferences.

## Main Purpose

This page helps the admin manage their own account without touching other users' data.

## What the Admin Sees

- Profile photo
- Name and role badge
- Phone and email
- Security settings
- Contact settings
- Preferences
- Sign out controls

## Step-by-Step Actions

### 1. Open My Profile
- The admin clicks **My Profile** from the sidebar.
- The page opens with the profile header.

### 2. Review account identity
- The admin sees their current name, phone, email, and role.
- If email is verified, the page shows the verification state.

### 3. Update profile photo
- The admin can change their avatar photo.
- The uploaded photo is shown in the profile header.

### 4. Change display name
- The admin can update first and last name.
- The new name is saved in the profile form.

### 5. Change password
- The admin opens the Security tab.
- They enter the current password and a new password.
- The page confirms when the password is updated.

### 6. Update contact information
- The admin can link or change email.
- The admin can change phone number through OTP verification.

### 7. Set preferences
- The admin can change theme and notifications.
- These settings help personalize the dashboard.

## Important Rules For Bemnet

- This page is only for the logged-in admin account.
- Do not describe other user management actions here.
- If the user asks about editing another person’s account, redirect to the correct admin section or to support.
