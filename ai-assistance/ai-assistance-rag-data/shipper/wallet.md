# Wallet

## What This Page Is

The Wallet page is the shipper's money area.

It shows available balance, wallet activity, pending withdrawals, withdrawal request forms, and withdrawal history.

## Main Purpose

This page lets the shipper manage their wallet balance and request payouts when allowed.

## What the Shipper Sees

- Available balance
- Total earned and total spent summary
- Pending withdrawal count
- Request Withdrawal button
- Withdrawal form
- Withdrawal request history
- Request status badges
- Approved amount and platform fee details
- User-submitted proof image
- Admin note and admin proof image when available
- Recent transaction summary

## Step-by-Step Actions

### 1. Open Wallet
- The shipper clicks **Wallet** in the sidebar.
- The wallet dashboard loads with the current balance and history.

### 2. Read the balance card
- The large balance number is the amount available now.
- The summary chips show total earned, total spent, and pending withdrawals.

### 3. Request a withdrawal
- If the wallet is not locked, the shipper clicks **Request Withdrawal**.
- A form appears below the balance card.

### 4. Fill the withdrawal form
The shipper enters:
- Amount to withdraw
- Bank name
- Account number
- Account holder name
- Payment method
- Optional note or message
- Optional proof file

### 5. Upload proof if needed
- The shipper can attach a receipt or image document.
- The preview shows before submission if it is an image.
- The file size limit must be respected.

### 6. Submit the request
- The shipper submits the form.
- If successful, a success state appears.
- The request then appears in the withdrawal history.

### 7. Read withdrawal history
- Each withdrawal card shows request amount, status, bank details, and date.
- Approved requests show approved amount and platform fee.
- The shipper can expand the card to see notes and attachments.

### 8. Understand locked wallet state
- If the wallet is locked, the shipper cannot create a new withdrawal request.
- A warning explains the lock reason if available.

## Important Rules For Bemnet

- Explain that withdrawal approval depends on admin review.
- Never tell the shipper that approval is instant if the system still requires review.
- If the wallet is locked, direct the shipper to support or staff.
- If a proof image is missing in history, explain that it should appear in the expanded withdrawal card when available.
- If the shipper asks about another user's wallet, refuse politely.
