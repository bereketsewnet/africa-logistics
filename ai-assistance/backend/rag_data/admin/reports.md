# Reports

## What This Section Is

The Reports section is where the admin views platform analytics.

## Main Purpose

It helps staff understand operational, financial, and logistics performance.

## What the Admin Sees

- Order report
- Finance report
- Driver report
- Logistics report
- Date range filters
- Charts and tables
- KPI summaries

## Step-by-Step Actions

### 1. Open Reports
- The admin opens the Reports section.
- Available report tabs appear based on permission.

### 2. Choose a report type
- The admin selects the report they need.
- Different tabs may include orders, finance, drivers, and logistics.

### 3. Set the date range
- The admin chooses the report period.
- The charts and summary update for that range.

### 4. Read the KPIs
- Each report shows the main numbers first.
- These help the admin spot trends quickly.

### 5. Inspect charts and tables
- The admin reviews visual breakdowns and recent rows.
- This helps support planning and decision-making.

## Important Rules For Bemnet

- Reports are for staff analysis, not public users.
- Do not invent platform-wide statistics.
- If the user asks for a report they do not have permission to view, explain that access is permission-based.
- Keep report explanations simple and tied to the visible metrics.
