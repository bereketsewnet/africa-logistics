# Wallet Adjustment

## What This Section Is

The Wallet Adjustment section is where the admin can add or subtract wallet value directly for corrective or operational reasons.

## Main Purpose

It lets staff fix wallet balance issues without using the normal shipper or driver wallet flows.

## What the Admin Sees

- Wallet adjustment form
- Target user selection
- Amount field
- Reason field
- Wallet balance summary
- Admin wallet tools if available

## Step-by-Step Actions

### 1. Open Wallet Adjustment
- The admin opens the wallet adjustment section.
- The adjustment form appears.

### 2. Choose the target user
- The admin selects the account that needs the change.
- This may be a shipper, driver, or other allowed user type.

### 3. Enter the amount
- The admin enters the correction amount.
- The system applies the adjustment according to the selected action.

### 4. Add a reason
- The admin provides a reason for the adjustment.
- This is important for audit tracking.

### 5. Submit the adjustment
- The admin submits the change.
- The wallet balance updates and the action is logged.

## Important Rules For Bemnet

- Wallet adjustment is a staff-only correction tool.
- Do not describe it as a normal user wallet action.
- Always mention that a reason or audit note is needed when explaining corrective actions.
- If the user asks to fix a wallet balance, direct them to the admin wallet adjustment workflow or support.
