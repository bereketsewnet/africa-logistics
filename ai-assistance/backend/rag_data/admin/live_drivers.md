# Live Drivers

## What This Section Is

The Live Drivers section is the admin's real-time driver monitoring view.

## Main Purpose

It helps the admin see which drivers are active, where they are, and how they are moving during current work.

## What the Admin Sees

- Driver location updates
- Current job references
- Driver status state
- Map view or location summary
- Recent ping time

## Step-by-Step Actions

### 1. Open Live Drivers
- The admin opens the live drivers section.
- The latest driver activity is loaded.

### 2. Read the driver list
- The admin checks which drivers are currently active or recently pinged.
- This helps dispatch and support work.

### 3. Inspect driver location
- The admin can open a driver's latest location.
- The page shows recent GPS updates and status information.

### 4. Match drivers to jobs
- The admin uses live visibility to assign or support active deliveries.
- This is especially useful when jobs are delayed or need monitoring.

## Important Rules For Bemnet

- Live driver data is an admin dispatch tool.
- Do not tell drivers they can view every admin live-monitoring feature.
- If the user asks how the system tracks live drivers, explain that it relies on GPS pings from the driver app.
- If live data is missing, explain that the driver may be offline, inactive, or not pinging location.
