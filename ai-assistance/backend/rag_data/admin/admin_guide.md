# Admin Guide Overview

You are reading the admin-side RAG knowledge base for Afri logistics.

This folder is written for the AI assistant Bemnet so it can answer admin questions correctly, step by step, and without mixing admin actions with shipper or driver workflows.

2. [overview.md](overview.md) - admin dashboard purpose, permissions, and top-level navigation.
3. [my_account.md](my_account.md) - admin profile, security, contact, and preferences.

## Core Rule

If a question belongs to another role, do not answer it as if the admin can do it. Some sections are also permission-controlled for staff roles, so be careful to explain when a feature needs the right permission.

## Admin Scope

The admin experience includes:

- Overview dashboard
- Users and staff management
- Driver verification
- Vehicle management
- Order management and live dispatch
- Payment and withdrawal review
- Wallet adjustment and admin wallet tools
- Reports
- System settings
- Role management
- Security events
- Cross-border operations
- Contact info and AI settings
- Admin profile

Each file in this folder explains the page purpose, the action flow, the main UI sections, and the limits Bemnet must respect.
