# Admin Overview

## What This Page Is

The Overview section is the admin dashboard home.

It shows system-level summaries, recent activity, key counts, and quick navigation into the rest of the admin tools.

## Main Purpose

This page is the starting point for staff who manage the logistics platform.

## What the Admin Sees

- System stats
- User lists
- Quick action cards
- Recent activity
- Shortcuts into drivers, shippers, staff, vehicles, orders, payments, reports, and settings
- Pending review counters such as payments and withdrawals

## Step-by-Step Actions

### 1. Open the dashboard
- The admin or staff user opens the admin area.
- The left sidebar and top bar appear.

### 2. Read the summary counts
- The dashboard shows totals and operational counters.
- These counters help staff understand what needs attention first.

### 3. Follow the quick actions
- The overview cards let the admin jump directly to major workflows.
- This reduces clicking and helps staff move faster.

### 4. Check pending work
- Pending payments, withdrawals, driver verifications, and other review queues may show badges.
- These badges are there to prioritize urgent tasks.

### 5. Open a section from the overview
- The admin clicks a card or menu item.
- The correct management section opens.

## Important Rules For Bemnet

- Explain the overview as a control center, not a public user page.
- If a user asks how to do staff work, check whether they actually have the permission.
- If they do not, tell them to contact the team that manages access.
- Do not invent dashboard counters or system totals.
